# Marketplace Mirakl-Powered — Catalogo Completo Europa

## Panoramica Generale

### Contesto Mirakl Globale

Mirakl è una società francese di software e-commerce che fornisce piattaforme marketplace in modalità SaaS (Software-as-a-Service) a retailer, produttori e grossisti. A livello mondiale, oltre **350 marketplace online** sono alimentati dalla tecnologia Mirakl, con un ecosistema composto da più di **100,000 venditori** distribuiti tra circa **450 operatori di marketplace**.

La piattaforma Mirakl è caratterizzata da:
- Gestione di oltre **600 milioni di SKU** (Stock Keeping Units)
- Elaborazione di **250 milioni di chiamate API** giornaliere
- Gestione di **1 miliardo di aggiornamenti di inventario** al giorno
- Crescita annuale del **34% nei marketplace Mirakl**, con performance **4 volte superiore** al settore e-commerce tradizionale

### Focus Europeo

In Europa, Mirakl alimenta i marketplace di alcuni dei maggiori retailer, tra cui:
- **Carrefour** (Francia, Spagna, Polonia, Belgio)
- **Fnac Darty** (Francia)
- **Leroy Merlin** (Francia, Italia, Spagna, Polonia, Belgio)
- **Decathlon** (Francia e altri paesi europei)
- **H&M** (marketplace a livello europeo)
- **El Corte Inglés** (Spagna, Portogallo)
- **Douglas** (Germania, Benelux, altri paesi europei)
- **Conrad** (Germania)
- **MediaMarkt/Saturn** (Germania, Austria, Spagna)
- **Galeries Lafayette** (Francia)
- **Maisons du Monde** (Francia)

### Caratteristiche Comuni Dei Marketplace Mirakl

Tutti i marketplace Mirakl condividono caratteristiche strutturali comuni:

**Architettura API Standardizzata:**
- Tutti utilizzano la **Mirakl Shop API** come struttura di base
- I pattern di integrazione API sono coerenti tra i vari marketplace
- La documentazione API tecnica rimane sostanzialmente simile

## 7. Caratteristiche Comuni dei Marketplace Mirakl

### 7.1 Processo Seller Onboarding

**Fase 1: Application (1-3 giorni)**
- Submission form
- Business info validation
- Catalogo preview

**Fase 2: Review (5-10 giorni)**
- Verifica documenti (SIREN, SIRET, VIES)
- Valutazione business model
- Contratto review

**Fase 3: Approval (1-2 giorni)**
- Approvazione management
- Setup account
- Accesso dashboard

**Fase 4: Go-Live (1 settimana)**
- Training seller
- First products integration
- Performance monitoring

### 7.2 Modello Commissioni Tipico

**Commissioni Standard:**
| Categoria | Commissione |
|-----------|-------------|
| Abbigliamento | 10-15% |
| Elettronica | 5-10% |
| Libri | 5% |
| Arredamento | 10-15% |
| Beauty/Cosmetici | 15-20% |
| Sport/Outdoor | 12-18% |

**Fee Aggiuntive:**
- Costo iscrizione: 0-500 EUR (una tantum)
- Costo commissione: 0-3% per transazione
- Logistics fee: 0-2 EUR per ordine

### 7.3 Requisiti Seller Comuni

**Documentazione:**
- VAT number europeo
- Registro commerciale
- Identification personale
- Documento indirizzo
- Bank account IBAN

**Business Requirements:**
- Catalogo minimo: 10-100 SKU (dipende marketplace)
- Rating seller: 4+ stelle (dopo prime vendite)
- Tempo risposta: <24 ore
- Return rate: <5%
- Tasso cancellazione: <2%

**Compliance:**
- GDPR compliance
- CCPA compliance (se applicabile)
- Consumer protection laws
- Tax registration
- Product safety certificates

### 7.4 Gestione Ordini Comune

**Order Lifecycle:**
```
Placed → Confirmed → Picked → Shipped → Delivered → Completed
   ↓        ↓         ↓        ↓        ↓         ↓
  Buyer      Seller      Seller     Carrier    Buyer      Review
 confirms   confirms    prepares   delivers   receives   (optional)
```

**Tracking Updates:**
- Automated email notifications
- Customer can track on marketplace
- Return label generation
- Delivery proof (signature)

### 7.5 Politiche di Return

**Standard Europeo:**
- 14 giorni di reso diritto del consumatore (EU directive)
- Reso senza motivo per online shopping
- Costo reso a carico di: dipende marketplace
- Return shipping label: buyer o seller provided

**Variazioni per Marketplace:**
| Marketplace | Return Policy |
|------------|---------------|
| Carrefour | 15 giorni, free return |
| FNAC Darty | 30 giorni, free return tech |
| Decathlon | 12 mesi, free return |
| H&M | 30 giorni, no questions asked |

### 7.6 Pagamenti e Settlement

**Ciclo Pagamenti:**
1. Customer paga marketplace
2. Marketplace holds payment (escrow)
3. Order delivered e confermato
4. Dispute window (es. 30 giorni)
5. Settlement a seller (es. settimanale)

**Settlement Schedule:**
| Marketplace | Frequenza | Delay |
|-------------|-----------|-------|
| Carrefour | Settimanale | 7-10 giorni |
| MediaMarkt | Bi-settimanale | 14 giorni |
| FNAC Darty | Mensile | 30 giorni |
| Decathlon | Settimanale | 10 giorni |

### 7.7 Strumenti Analytics

**Metrics Disponibili:**
- Sales (number, value)
- Conversion rate
- Visitor traffic
- Product performance
- Inventory levels
- Returns ratio
- Customer ratings
- Cancellation rate

**Export Options:**
- CSV export
- API access
- Real-time dashboards
- Report scheduling
