### 1.2 Customs Union & No Tariffs Intra-EU

#### EU Customs Union Characteristics

The EU Customs Union established a unified tariff system eliminating all internal tariffs between member states.

| Characteristic | Details |
|---|---|
| Internal Tariffs | 0% on all goods (except restricted items) |
| External Tariff | Common External Tariff (CET) applied uniformly |
| VAT Treatment | Reverse charge mechanism for B2B; standard VAT at destination |
| Excise Duties | Harmonized rates on fuel, alcohol, tobacco |
| Quota System | No quotas on intra-EU trade |
| Rules of Origin | EU rules (cumulation within EU) |
| Documentation | Simplified (CRM for road only if required) |

#### Restricted Items Within EU

Certain goods require special documentation and approval even for intra-EU movement:

### 1.3 Documentation Requirements for Intra-EU Trade

#### CMR (Convention on the Contract for the International Carriage of Goods by Road)

**Status:** Mandatory for all road freight within EU, EFTA, and associated countries.

| Document Element | Requirements | Consequences of Non-Compliance |
|---|---|---|
| CMR Consignment Note | 3 copies (carrier, sender, consignee); signed by carrier | €100-500 fine per missing copy |
| Goods Description | Detail, quantity, packaging, weight | Cargo seizure, delay of 2-5 days |
| Pickup & Delivery Addresses | Full street address, postal code, contact | Delivery failure, return to origin |
| Carrier Signature | Mandatory before departure | Cargo not covered under CMR protection |
| Weight Declaration | Actual gross weight in kg | Overweight fines: €0.05-0.10 per kg excess |
| Hazmat Declaration | If applicable, hazmat class, UN code | €500-2,000 fine for non-declaration |

**Transit Time Impact of CMR Non-Compliance:** +1-3 days due to border checks and administrative processing.

#### Packing List

**When Required:**
- B2B shipments with multiple line items (5+)
- Mixed product categories
- Wholesale shipments
- High-value goods (>€5,000)

**Mandatory Fields:**
- Packing list number and date
- Sender and consignee full details
- Product description (SKU, quantity, unit price)
- Total value and weight
- Special handling instructions

#### Commercial Invoice

**Requirements:**
- For non-EU shipments only
- Invoice date and number
- Detailed product description with tariff codes
- Unit price and total value
- Currency and payment terms
- Incoterms 2020
- VAT treatment declaration

## 2. CUSTOMS & IMPORT/EXPORT PROCEDURES

### 2.1 EU Common External Tariff (CET)

The CET is the unified tariff applied by all EU member states to goods entering from non-EU countries.

#### Tariff Structure

The EU Customs Tariff (based on HS 2022) contains:
- **10,000+ commodity codes** with specific duty rates
- **Duty rates:** 0% (raw materials, many industrial goods) to 300% (agricultural products)
- **Average applied rate:** 5.2% across all product categories

#### Common Tariff Rate Examples (2026)

| Product Category | HS Code | Duty Rate | AV/Ad-Valorem | Special Conditions |
|---|---|---|---|---|
| Raw silk | 5001 | 0% | Free | - |
| Electrical machinery | 8501-8517 | 0-4.7% | Varies by complexity | - |
| Footwear (leather) | 6403 | 8-12% | Per pair | - |
| Sugar | 1701 | €339-422/ton | Specific | Quota system |
| Wine (in containers >2L) | 2204 | €0.68-1.33/L | Specific + Ad-val | Trade agreement rates |
| Dairy products | 0401-0406 | 36-368% | Specific/Ad-val | TRQ (Tariff Rate Quota) |
| Fresh beef | 0201 | €304-378/ton | Specific | TRQ system |
| Cereals | 1001-1005 | €95-135/ton | Specific | Variable levy system |
| Clothing (cotton) | 6204 | 12% | Ad-valorem | Rules of origin (yarn forward) |
| Steel products | 7208-7326 | 0-4.5% | Ad-valorem | Safeguard duties possible |

#### Preferential Trade Agreements Affecting Tariffs

| Agreement | Partners | Duty Rate | Scope |
|---|---|---|---|
| GSP (Generalized Scheme of Preferences) | 104 developing countries | 0-15% | Full duty elimination to 50% reduction |
| CETA (Canada) | Canada | 0% (majority) | 98.7% duty-free immediately |
| MERCOSUR (under negotiation) | Argentina, Brazil, Paraguay, Uruguay | TBD | Target: 92% duty-free |
| UK (TCA Post-Brexit) | United Kingdom | 0% (Rules of Origin apply) | Tariff-free with RoO compliance |
| EFTA | Switzerland, Norway, Iceland, Liechtenstein | 0-10% | Industrial goods mostly duty-free |

### 2.2 TARIC Code System

**TARIC (Tariff Integrated Computerized Research)** is the EU's 10-digit tariff code system.

#### TARIC Code Structure

```
Example: 6204 62 10 10 90
├─ HS 6-digit: 620462 (Women's cotton trouser-type garments)
├─ EU 8-digit: 62046210 (Woven cotton, not stretch, industrial use)
├─ TARIC 10-digit: 6204621090 (Specific preference/regulation)
```

**Components:**
- **HS 6-digit (Harmonized System):** Global standard, ~5,000 categories
- **HS 8-digit:** EU classification adds detail, ~14,000 codes
- **TARIC 10-digit:** Full EU detail including duties, measures, quotas, exemptions

#### TARIC Data Elements

| Data Field | Example | Purpose |
|---|---|---|
| Commodity Code | 6204621090 | Unique product identifier |
| Description | Women's trouser-type garments of cotton | Product classification |
| General Duty | 12% | Standard tariff rate |
| Additional Duty | 0% (or specific) | Supplementary charges |
| Preferential Duties | 0%, 5%, 8% | Depends on origin country/FTA |
| Antidumping Duties | 25-45% | On specific countries/producers |
| Safeguard Measures | Applied if triggered | Temporary protection |
| Exemption Codes | Code 200, 300, etc. | Qualifies for zero-duty entry |
| National Origin Mark | Yes/No | Country of origin marking required |
| Quotas | In-quota rate/out-of-quota | Tariff Rate Quota (TRQ) applies |

#### How to Find TARIC Codes

**Resources:**
1. **TARIC Online Database:** https://ec.europa.eu/taxation_customs/business/calculation-customs-duties/what-is-taric_en
2. **ASYCUDA++:** Used by customs brokers for classification
3. **HS Database:** WCO (World Customs Organization) base codes
4. **Binding Tariff Information:** BTI (valid 6 years, €300-500 fee)

**Cost to Obtain Binding Tariff Information:**
- Fee: €300-500 per classification request
- Processing time: 60-120 days
- Validity: 6 years from issuance
- Advantage: Legally binding classification defense if origin/product unchanged

### 2.3 Customs Procedures

#### Normal (Formal) Customs Procedure

**When Used:** All imports/exports requiring full documentation review.

**Steps & Timeline:**

| Step | Timeline | Documents | Cost |
|---|---|---|---|
| 1. Customs Declaration Lodging | Day 0 | Entry/Exit Decl., Commercial Invoice, TARIC | Included in procedure |
| 2. Risk Assessment | Day 0-1 (48 hrs) | Customs review of risk profile | Included |
| 3. Documentary Check | Day 1-2 | Full document verification | Included |
| 4. Physical Inspection (if triggered) | Day 2-5 | Sample/full cargo examination | €50-200 (inspection fee) |
| 5. Duty Assessment & Calculation | Day 3-5 | Apply TARIC, compute duties | Included |
| 6. Clearance Grant | Day 5-7 | Release permit, goods cleared | Goods released |
| 7. Goods Release | Day 6-7 | Delivery to consignee | - |

**Total Time:** 5-7 working days for standard shipment; 2-3 days if no inspection triggered.

**Costs (per shipment, €2,000-5,000 value):**
| Cost Component | Amount |
|---|---|
| Customs Broker fee | €100-200 |
| Customs formalities handling | Included |
| Inspection fee (if any) | €50-150 |
| Duty (at 10% avg) | €200-500 |
| **Total | €350-850 |

#### Simplified Customs Procedure

**Eligibility:** Authorized Economic Operator (AEO) status OR pre-approved low-risk trader.

**Requirements:**
- Annual import/export volume: Minimum €500,000
- Compliance record: 99%+ rate
- Clean audit history: 2+ years
- IT systems certified

**Advantages:**
- Timeline reduction: 1-2 days vs. 5-7 days
- Cost reduction: 40-50% lower broker fees
- Documentation reduction: Simplified entry form sufficient
- Risk rating: Lower profile, fewer inspections (0.5% vs. 5% for non-AEO)

**Example Timeline (AEO):**
- Day 0: Lodging + Risk check (simultaneous)
- Day 1: Clearance grant (no physical check)
- Day 1-2: Goods released

**Cost Benefit:**
| Item | Normal | AEO |
|---|---|---|
| Processing time | 5-7 days | 1-2 days |
| Broker fees | €150-250 | €80-120 |
| Inspection risk | 5-8% | 0.5-1% |
| Avg clearance cost | €400-600 | €200-300 |
| Annual fee for AEO status | N/A | €1,000-2,000 (varies by port/authority) |

#### Transit Customs Procedure (TIR Carnet, Common Transit)

**Used for:** Goods passing through multiple countries en route to final destination.

**Options:**

| Procedure | Scope | Coverage | Bond Required | Valid For |
|---|---|---|---|---|
| TIR Carnet | Non-EU to Non-EU via EU | UN TIR Convention | €50,000-500,000 | All signatories |
| Common Transit (CT) | EU, EFTA (4 countries) | EU-EFTA corridor | €5,000-50,000 | EU+EFTA only |
| NCTS (New Computerized Transit System) | EU internal + EFTA | All EU-EFTA borders | €5,000-30,000 | EU+EFTA, digital |

**TIR Carnet Details:**
- **Validity:** 1 year (can be renewed 9 times, total 10 years)
- **Cost:** Approx. €500-1,500 per carnet (includes bond guarantee)
- **Coverage:** Full value of goods + duty risks
- **Customs seals:** Applied at origin, verified at destination
- **Time saving:** Skip documentary checks at intermediate borders

**Example TIR Route & Costs:**
- Route: China (Port) → Germany (Port) → France (Customer)
- Carnet cost: €800
- Sealing at Hamburg: €50
- Verification at Lyon: €30-50 (usually waived for TIR)
- Total transit cost: €880-900
- Without TIR: Documentary checks at Germany + France border (~€200 extra, +1-2 days)

### 2.4 Customs Broker Selection

#### Role of Customs Broker

A customs broker is a licensed professional representing importers/exporters before customs authorities.

**Responsibilities:**
1. Prepare customs documentation (Entry, Exit, Transit declarations)
2. Calculate duties and determine correct TARIC codes
3. Handle all correspondence with customs
4. Arrange inspections (if needed)
5. Ensure compliance with regulations
6. Manage post-clearance audits

**When Required:**
- All non-EU imports/exports (mandatory in most EU countries)
- Complex classifications (fragile, dangerous, restricted goods)
- First-time importers
- High-value shipments (€5,000+)

### 2.5 Customs Value Calculation

**Customs value** is the basis for duty assessment; incorrect valuation = underpayment risk + penalties.

#### Methods (Prioritized Under WCO Valuation Agreement)

| Method | Priority | Definition | Example |
|---|---|---|---|
| **Transaction Value** | 1 (Primary) | Price paid for goods + delivery costs to border | $100 invoice + $10 freight = $110 CIF |
| **Identical Goods** | 2 | Value of identical goods in comparable sales, same time | Same product from different supplier to same EU port |
| **Similar Goods** | 3 | Value of similar (not identical) goods | Different color/size, same product type |
| **Deductive Method** | 4 | Resale price in EU market minus profit + costs | €150 retail price - 30% margin - €10 import costs = €95 value |
| **Computed Value** | 5 | Producer cost + profit margin (negotiated or standard) | €50 cost + 8% profit = €54 value |
| **Fallback Value** | 6 (Last Resort) | Reasonable basis using available info | Historical avg. price for category |

#### Included/Excluded Elements in Customs Value

| Element | Included? | Notes |
|---|---|---|
| **Invoice price** | YES | Base transaction value |
| **Freight to EU border** | YES | CIF (Cost, Insurance, Freight) to EU port |
| **Insurance | YES | Ocean/air insurance premiums |
| **Royalties/License fees** | YES | If condition of sale (e.g., patent royalty) |
| **Assists (materials provided)** | YES | Buyer-supplied materials, tools, dies |
| **Internal transport (in origin country)** | YES | Factory to port in origin |
| **Packaging (product packaging)** | YES | Boxes, wrapping included with goods |
| **Rebates/Discounts (post-sale)** | NO | Applied after valuation |
| **Transport within EU** | NO | Not included (internal EU movement) |
| **Duties in origin country** | NO | Already paid in non-EU origin |
| **After-sales service costs** | NO | Warranty, installation in EU excluded |
| **Financing costs** | NO | Interest, if financing arranged |

#### Customs Value Calculation Example

**Scenario:** Import 100 units electronic devices from Vietnam.

**Calculation:**
| Component | Amount | Calculation |
|---|---|---|
| Invoice price (100 × $50) | $5,000 | - |
| Factory-to-port transport (Vietnam) | $200 | 4% of invoice |
| Ocean freight (port-to-port) | $450 | Standard rate Vietnam-Rotterdam |
| Insurance | $35 | 1% of (invoice + freight) |
| **Customs Value (CIF)** | **$5,685** | Sum of above |
| EUR conversion (1 USD = 0.92 EUR) | €5,230 | - |
| **Duty rate (TARIC: 10%)** | €523 | 10% × €5,230 |
| **VAT (21% on CIF + duty)** | €1,198 | 21% × (€5,230 + €523) |

**Total duties + VAT: €1,721** (approximately 30% of CIF value)

**Post-Clearance Audit Risk:**
- Customs may audit if invoice price is 20%+ below market average
- If value deemed underestimated: Additional duty + 10-25% penalty + interest
- Scenario: If audited and value reassessed to $60/unit (market rate), additional duty = €104; penalty = €26; total exposure = €130

#### Advanced Valuation Techniques

**Royalty Adjustment:**
If goods have embedded intellectual property:
- Patent licensing fee (2-5% of transaction value): Must be added
- Example: Electronics with patented circuit design (5% royalty) = +€260 value
- Must declare in customs entry if applicable

**Advance Pricing Agreement (APA):**
- Request pre-agreement with customs on valuation for 3-5 years
- Fee: €1,500-3,000; processing time: 90-180 days
- Benefit: Certainty, reduced audit risk for repetitive imports
- Available through EU Customs Authorities

### 2.6 Incoterms 2020 (International Commercial Terms)

Incoterms define responsibilities for goods transfer, costs, and risks between buyer and seller.

#### Seven Main Incoterms for International Trade

| Incoterm | Full Name | Responsibility Transfer | Shipping Cost | Insurance Cost | Risk Transfer | When to Use |
|---|---|---|---|---|---|---|
| **EXW** | Ex Works | At seller's location | Buyer | Buyer | Buyer from collection | Small shipments; buyer controls logistics |
| **FCA** | Free Carrier | At named place (usually port) | Seller to carrier | Buyer | Buyer from handoff | Container LCL; buyer arranges consolidation |
| **CPT** | Carriage Paid To | To named place | Seller | Buyer | Buyer from shipment | Any mode; seller pays freight only |
| **CIP** | Carriage + Insurance Paid | To named place | Seller | Seller | Buyer from shipment | Multi-modal; seller provides insurance |
| **DAP** | Delivered At Place | To final destination (unloaded) | Seller | Buyer | Buyer at destination | Land transport within EU; final destination named |
| **DDP** | Delivered Duty Paid | To final destination | Seller | Buyer | Seller (at destination, all duties paid) | Full responsibility seller; buyer receives cleared goods |

#### Incoterms 2020 Cost Impact (€100 CIF Value Shipment)

| Incoterm | Seller Pays | Buyer Pays | Effective Cost to Buyer | Best For |
|---|---|---|---|---|
| EXW | €0 (FOB extra) | €140-180 | €140-180 | Direct importers with logistics |
| FCA | €110 | €30-40 | €140-150 | Consolidation buyers |
| CPT | €130 | €20-30 | €150-160 | Pre-arranged freight |
| CIP | €130 + €5 insurance | €15-25 | €150-160 | Insurance-conscious buyers |
| DAP | €170 | €5-10 duty/VAT | €175-180 | Typical EU imports |
| DDP | €180-200 | €0 | €180-200 | Retail, complete delivery |

#### Incoterms Selection for EU Imports (Practical Guide)

**From Non-EU (e.g., China to Germany):**
- **Importer typically uses:** DAP (Delivered At Place, named German warehouse)
- **Reason:** Seller manages all shipping risks to destination; importer clears customs
- **Cost:** Typically 20-25% premium over EXW
- **Alternative if importer is AEO:** CPT (Carriage Paid) to allow direct control of customs clearance

**From EU to EU (Intra-EU movement):**
- **Standard practice:** DDP (Delivered Duty Paid) - no duties apply
- **Cost:** Shipping cost only (no duty/VAT since intra-EU)
- **Alternative:** EXW if buyer (distributor) arranges pickup for consolidation

**From UK to EU (Post-Brexit):**
- **Recommended:** CIP (Carriage + Insurance Paid to border) or DAP (to EU warehouse)
- **Reason:** Customs clearance required; seller controls shipment integrity
- **Cost impact:** +15-20% vs. pre-Brexit due to additional customs handling

### 2.7 Authorized Economic Operator (AEO) Status

**AEO** is a certification granted by customs authorities to companies meeting compliance standards.

#### AEO Benefits & Requirements

| Benefit | Value | Requirement |
|---|---|---|
| **Faster customs clearance** | 50-75% time reduction | Clean 24+ month compliance record |
| **Reduced inspections** | 0.5-1% vs. 5-8% | Annual certification audit |
| **Simplified procedures** | Standard documentation sufficient | Advanced IT systems (EDI/API capable) |
| **Priority processing** | Dedicated customs officer | €1,000-3,000 annual fee |
| **Mutual recognition** | Some non-EU countries (Japan, Singapore) | Compliance with EU PCS (Authorized Consignee) |
| **Supply chain visibility** | Reduced border delays for partners | Regular submission of records |

#### AEO Application Process

**Timeline:** 4-6 months

| Phase | Duration | Key Activities |
|---|---|---|
| **Pre-assessment** | 4-6 weeks | Customs agency evaluation; document preparation |
| **Full audit** | 8-12 weeks | Site visit; financial records; IT systems review |
| **Decision** | 2-4 weeks | Final approval/rejection decision |
| **Certification** | Upon approval | 3-year validity; annual compliance reports |

**Cost Structure (EU-wide):**
| Cost Element | Amount |
|---|---|
| Application fee | €500-1,500 |
| Audit cost (internal staff) | €2,000-4,000 |
| System upgrades (if needed) | €5,000-15,000 |
| Annual certification fee | €800-2,500 |
| **Total first year | €8,300-23,000 |

**ROI Calculation (Annual):**
- **Time savings:** 3 days per shipment × 1,000 shipments/year = 3,000 days
- **Value:** 3,000 days × €50/day (staff time + broker) = €150,000 savings
- **Cost:** €2,300/year (avg. maintenance)
- **Net ROI:** 65× payback (typically 1-2 months)

#### AEO Maintenance Requirements

**Annual Compliance Obligations:**
1. Submit customs records (monthly reports)
2. Maintain clean compliance history (≥99% accuracy)
3. Staff training on customs regulations (2 sessions/year)
4. Conduct internal audits (quarterly)
5. System security/IT updates (ongoing)
6. Financial solvency maintenance

**Risk of Losing AEO:**
- Non-compliance incidents: 3+ violations in 12 months = suspension review
- Penalty: 6-12 month suspension, then reapplication required
- Cost of reapplication: €3,000-5,000
