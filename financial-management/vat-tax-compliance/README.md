## Quick Actions

1. Register OSS (Agenzia delle Entrate) if cross-border sales > €10,000/year
2. Register IOSS if importing goods ≤€150
3. Install Taxdoo (€99-€299/month) for quarterly compliance
4. Set calendar reminders: Apr 20, Jul 20, Oct 20, Jan 20 (OSS deadlines)
5. Reconcile DAC7 data (June 30) against OSS returns before July 31
6. Maintain 10-year record retention system
