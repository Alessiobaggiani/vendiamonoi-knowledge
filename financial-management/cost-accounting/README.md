## Quick Reference - Core Formulas

**Contribution Margin:** CM/unit = Selling_Price - COGS - Marketplace_Fees - Fulfillment - Advertising

**Break-Even:** Break_Even_Units = Fixed_Costs ÷ CM_per_unit

**Landed Cost:** LC/unit = (Product + Freight + Customs + VAT + Handling + Packaging) ÷ Units
