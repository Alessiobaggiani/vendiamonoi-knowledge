# ADR-004: Sync obbligatoria GitHub + Obsidian

**Status:** Accepted
**Date:** 2026-04-15

## Decision
Ogni documentazione va pushata automaticamente su GitHub E Obsidian.
Notion solo su richiesta esplicita.

## Sync procedure
1. Scrivi contenuto UNA volta
2. Push su GitHub (vendiamonoi-knowledge) via MCP
3. Write su Obsidian via MCP
4. In un unico passaggio
