# Cowork Error Registry

Registro errori permanente per Claude Cowork. Ogni errore diventa una regola operativa permanente.

## Struttura
```
error-registry/
  sellrapido/
    errors_log.md
    rules.md
  make/
    errors_log.md
    rules.md
  notion/
    errors_log.md
    rules.md
  supabase/
    errors_log.md
    rules.md
  github/
    errors_log.md
    rules.md
  api/
    errors_log.md
    rules.md
```

## Principio guida
> Prima di eseguire, consulta le regole del software coinvolto. Dopo ogni errore, aggiorna il registro e consolida la lezione in una regola permanente.
